# Инструкция по обновлению фотографий и данных картин

Данный документ предназначен для автоматизации процесса добавления новых работ Ирины Барабаш на сайт. Следуйте этим шагам при каждом обновлении.

## 1. Подготовка изображения
- **Источник:** Новые фотографии обычно присылаются пользователем или скачиваются с профиля Singulart.
- **Множественные изображения:** У каждой картины на Singulart обычно есть несколько фото (главное, детали, вид в интерьере). **Обязательно сохранять все доступные ракурсы** для каждой работы.
- **Обработка:** Главное изображение должно быть обрезано строго по краям холста. Дополнительные фото (детали) сохраняются как есть.
- **Формат:** JPG или PNG.
- **Путь сохранения:** `public/images/artworks/`. Рекомендуется создавать подпапку для каждой картины, если изображений больше одного (например, `public/images/artworks/slug/`).
- **Именование:** Главное фото — `main.jpg`, дополнительные — `detail-1.jpg`, `detail-2.jpg` и т.д.

## 2. Обновление базы данных (Artworks Data)
- **Файл:** `src/data/artworks.ts` (или аналогичный файл, где хранится массив `artworks`).
- **Действие:** Добавить новый объект в массив или обновить существующий.
- **Поля:**
  - `id`: Уникальный строковый идентификатор (slug).
  - `title`: Название картины.
  - `year`: Год создания.
  - `medium`: Техника (например, "Oil on Canvas").
  - `dimensions`: Размеры (например, "60 x 80 cm").
  - `price`: Цена (если применимо).
  - `image`: Путь к файлу `/images/artworks/filename.jpg`.

## 3. Локализация (Переводы)
- **Файлы:**
  - `src/i18n/dictionaries/ru.json`
  - `src/i18n/dictionaries/en.json`
- **Действие:** Если названия картин хранятся в словарях, добавьте ключи для новой работы в оба файла. Убедитесь, что ключи совпадают.

## 4. Проверка на сайте
1. Запустите локальный сервер: `npm run dev`.
2. Перейдите в галерею или на страницу новой категории.
3. Проверьте отображение в режиме «Примерка в интерьере» (Room Preview), убедившись, что масштабы (размеры в см) указаны верно.

## 5. Аудит (Singulart Sync)
Периодически проверяйте соответствие списка в `artworks.ts` актуальному профилю художника на Singulart. При обнаружении пропусков — скачайте недостающие фото и повторите шаги 1-3.
